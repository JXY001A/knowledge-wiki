# 快速笔记

保存闪念笔记到助理数据库，自动提取标签并建立 FTS5 全文索引。

## 触发条件

- 用户输入包含：笔记、记一下、备忘、记录、备注、保存

## 输入

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| input_text | string | ✅ | 笔记内容 |
| user_id | string | ✅ | 用户 ID |
| send_md | function | ✅ | 回复回调 |

## 输出

确认消息含笔记摘要和自动提取的标签。

## 执行步骤

1. 提取笔记正文（去除触发词）
2. 用 Ollama 自动提取标签
3. 写入 notes 表 + FTS5 索引
4. 回复确认

## 示例

输入: "笔记 部署脚本 health check 超时设置 #DevMechin"
输出: "📝 已保存笔记 [DevMechin, 运维]"
