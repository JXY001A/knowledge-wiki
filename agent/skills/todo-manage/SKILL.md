# 待办管理

管理个人待办事项：创建、查询、完成、删除。支持优先级、截止日期、标签。

## 触发条件

- 用户输入包含：待办、todo、任务、完成、取消、要做、做

## 输入

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| input_text | string | ✅ | 用户输入文本 |
| user_id | string | ✅ | 企业微信用户 ID |
| send_md | function | ✅ | 回复回调 |

## 输出

通过 send_md 发送格式化的待办操作结果。

## 执行步骤

1. 用 Ollama 解析用户意图（create/list/complete/delete）
2. 调用 assistant/models 的 CRUD 操作
3. 格式化结果并通过 send_md 回复

## 示例

输入: "待办 明天前提交Q2报告 高优先级"
输出: "✅ 已创建待办：提交Q2报告 [高优先级] [截止: 2026-06-08]"
