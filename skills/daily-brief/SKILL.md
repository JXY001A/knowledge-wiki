# 每日简报

生成每日早报或晚报：汇总今日待办、提醒和知识库动态。

## 触发条件

- 用户输入包含：日报、早报、晚报、今天有什么、简报、总结
- 调度器定时触发（早上 8 点和晚上 18 点）

## 输入

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| input_text | string | ✅ | 用户输入或调度触发 |
| user_id | string | ✅ | 企业微信用户 ID |
| send_md | function | ✅ | 回复回调 |

## 输出

通过 send_md 发送格式化的每日简报（Markdown）。

简报内容：
- 📋 今日待办（未完成 + 今日到期）
- ⏰ 今日提醒
- 📝 今日笔记
- 📚 知识库更新（新增/更新页面）

## 前置依赖

- `assistant.scheduler` — 定时调度器
- `daily_brief` — MCP 工具
