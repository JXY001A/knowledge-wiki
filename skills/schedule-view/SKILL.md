# 日程查看

查看今日（或指定日期）的日程安排：待办事项、提醒和习惯打卡状态。

## 触发条件

- 用户输入包含：日程、今天要做什么、今天有什么、明天、这周、安排

## 输入

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| input_text | string | ✅ | 用户输入 |
| user_id | string | ✅ | 企业微信用户 ID |
| send_md | function | ✅ | 回复回调 |

## 输出

通过 send_md 发送格式化的日程视图（Markdown）：

- 📋 待办列表（按优先级排列）
- ⏰ 提醒列表（按时间排列）
- ✅ 今日习惯打卡状态

## 前置依赖

- `assistant.db` — SQLite 待办/提醒/习惯存储
- `schedule_today` — MCP 工具
