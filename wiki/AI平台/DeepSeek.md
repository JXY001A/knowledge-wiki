> 中国 AI 公司深度求索（DeepSeek）的 API 平台，提供 V4 系列大语言模型服务，兼容 OpenAI 与 Anthropic 两种 API 格式。

## 概述

DeepSeek 由深度求索公司运营，提供 deepseek-v4-flash（轻量）和 deepseek-v4-pro（旗舰）两个模型，均具备 1M 上下文窗口、思考模式、JSON 输出和 Tool Calls 能力。API 同时兼容 OpenAI 格式和 Anthropic 格式。

## 关键贡献 / 特性

- 1M 上下文窗口（两个模型均为标配）
- 384K 最大输出长度
- 双 API 格式兼容（OpenAI + Anthropic）
- 思考模式（Thinking Mode）支持
- 缓存命中价格低至 0.02 元/百万 tokens
- FIM（Fill-in-the-Middle）补全（Beta）

## 时间线

- 2026/4/26：全系列缓存命中价格降至首发价 1/10
- 2026/05/31：V4-Pro 2.5 折优惠到期
