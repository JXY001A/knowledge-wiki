## 二、企业微信消息处理全链路

```mermaid
flowchart TD
    WX["手机微信"] -->|"用户发消息"| WXServer["企业微信服务端"]
    WXServer -->|"POST XML（AES 加密）"| Frps["阿里云 frps<br/>8.133.175.201:9400"]
    Frps -->|"frp 隧道"| Frpc["DevMechin frpc<br/>→ 127.0.0.1:9400"]
    Frpc -->|"转发"| Flask["Flask receive_message()"]

    Flask -->|"1. 提取加密 XML"| Decrypt["decrypt_msg()<br/>AES-256-CBC 解密"]
    Decrypt -->|"2. 明文 XML"| Parse["解析 MsgType<br/>FromUserName<br/>Content"]
    Parse -->|"3. 后台线程"| Thread["threading.Thread<br/>target=process_message"]

    Thread --> Dispatch["process_message()<br/>Skill 路由引擎"]

    Dispatch -->|"? + 搜索"| SkillSearch["search-wiki"]
    Dispatch -->|"? + 查询"| SkillQuery["query-knowledge"]
    Dispatch -->|"URL"| SkillIngest["ingest-article"]
    Dispatch -->|""检查/lint""| SkillLint["lint-wiki"]
    Dispatch -->|"其他文本"| SkillSave["save-note（默认回退）"]

    SkillSearch --> Reply["send_markdown()<br/>→ 企业微信 API"]
    SkillQuery --> ReplyCards["send_template_card()<br/>多卡片 + markdown"]
    SkillIngest --> Reply
    SkillLint --> Reply
    SkillSave --> Reply

    Reply --> WXAPI["qyapi.weixin.qq.com<br/>主动消息 API"]
    ReplyCards --> WXAPI
    WXAPI -->|"推送"| WX
```

## 三、Skill 执行引擎内部流程

```mermaid
flowchart TD
    Start["process_message(user_id, text, send_md, send_tpl)"] --> Strip["stripped = text.strip()"]
    Strip --> BuildCtx["构建 context 字典<br/>{user_id, input_text, send_md, send_tpl}"]

    BuildCtx --> Check{"text 以 ? 开头?"}

    Check -->|"是"| ExtractQ["question = stripped[1:]<br/>ctx['query'] = question"]
    ExtractQ --> MatchQ["match_skill(question)"]

    Check -->|"否"| CheckURL{"is_url(text)?"}
    CheckURL -->|"是"| MatchURL["match_skill(url)"]
    CheckURL -->|"否"| MatchText["match_skill(text)"]

    MatchQ --> Found{"skill 匹配成功?"}
    MatchURL --> Found
    MatchText --> Found

    Found -->|"是"| Exec["execute_skill(skill.name, ctx)"]
    Found -->|"否"| Fallback{"回退链"}

    Fallback -->|"?"| FBQuery["execute_skill('query-knowledge', ctx)"]
    Fallback -->|"URL"| FBIngest["execute_skill('ingest-article', ctx)"]
    Fallback -->|"text"| FBSave["execute_skill('save-note', ctx)"]

    Exec --> LoadSkill["1. find_skill(name)<br/>→ Skill 对象"]
    FBQuery --> LoadSkill
    FBIngest --> LoadSkill
    FBSave --> LoadSkill

    LoadSkill --> LoadImpl["2. _load_impl(skill)<br/>importlib 加载<br/>skills/{name}/impl.py"]
    LoadImpl --> HasImpl{"impl.py 存在?"}

    HasImpl -->|"是"| CallImpl["3. impl.execute(context)"]
    HasImpl -->|"否"| FallbackMD["返回 SKILL.md 文本<br/>+ 提示缺少实现"]

    CallImpl --> Async{"async def execute?"}
    Async -->|"是"| Await["asyncio.run(execute(ctx))"]
    Async -->|"否"| Sync["execute(ctx)"]

    Await --> Result["返回结果文本"]
    Sync --> Result
    FallbackMD --> Result

    Result --> SendBack{"结果非空?"}
    SendBack -->|"是"| SendMD["send_md(user_id, result[:3000])"]
    SendBack -->|"否（技能已回调发送）"| Done["处理完成"]
    SendMD --> Done
```

## 四、URL 摄取完整流水线（ingest-article）

```mermaid
flowchart TD
    URL["用户发送 URL"] --> Notify["send_md('正在提取并分析...')"]

    Notify --> Fetch["1. fetch_url_text(url)"]
    Fetch --> FetchDetail["urllib GET<br/>User-Agent: KnowledgeBot<br/>提取 og:title / h1 / title<br/>去除 script/style/nav<br/>限制 50KB"]
    FetchDetail --> FetchOK{"下载成功?"}

    FetchOK -->|"否"| SaveRef["save_to_inbox(待摄取链接)<br/>git commit + push"]
    SaveRef --> NotifyFail["send_md('无法下载...')"]

    FetchOK -->|"是"| SaveRaw["2. save_to_inbox(raw_text)<br/>→ raw/收件箱/wecom-url-{ts}.md"]

    SaveRaw --> DeepSeek["3. call_ingest(raw_text, url)"]
    DeepSeek --> DeepSeekDetail["POST api.deepseek.com/v1/chat/completions<br/>model: deepseek-v4-pro<br/>system: INGEST_SYSTEM_PROMPT<br/>temperature: 0.1 · max_tokens: 4096<br/>content: raw_text[:40000]"]

    DeepSeekDetail --> DeepSeekOK{"LLM 返回 JSON?"}

    DeepSeekOK -->|"否"| NotifyLLMFail["send_md('LLM 分析失败...')"]

    DeepSeekOK -->|"是"| ParseJSON["json.loads(响应)<br/>→ {title, domain, tags, summary,<br/>   key_points, notes, quotes,<br/>   concepts, related_pages}"]

    ParseJSON --> BuildSource["4. build_source_page(data, url)<br/>→ wiki/资料摘要/{title}.md<br/>frontmatter + 正文结构"]

    BuildSource --> BuildConcepts["5. build_concept_page(concept)<br/>→ wiki/概念/{name}.md<br/>（跳过已存在的）"]

    BuildConcepts --> AppendLog["6. append_ingest_log(data, url, title)<br/>→ wiki/操作日志.md"]

    AppendLog --> GitPush["7. commit_and_push(msg)<br/>git add -A → commit → push"]

    GitPush --> NotifyOK["send_md('✅ 已摄取到知识库<br/>标题 + 领域 + 摘要 + 新概念')"]
```

## 五、知识查询流水线（query-knowledge）

```mermaid
flowchart TD
    Q["用户: ? AI Workflow 是什么"] --> MCP["1. mcp_query(question)"]

    MCP --> MCPDetail["初始化 MCP 会话<br/>POST :9300/mcp → initialize<br/>获取 Mcp-Session-Id<br/>POST tools/call → query"]
    MCPDetail --> MCPOK{"找到结果?"}

    MCPOK -->|"否"| SearchFallback["回退: tools/call → search"]
    SearchFallback --> SearchFallbackOK{"找到?"}
    SearchFallbackOK -->|"否"| NotFound["send_md('暂无相关内容')"]
    SearchFallbackOK -->|"是"| ReadPages["读取匹配页面全文<br/>strip_frontmatter<br/>限制 2000 字/页"]

    MCPOK -->|"是"| StripFM["2. strip_frontmatter(context)<br/>去除 YAML 元数据"]

    ReadPages --> StripFM

    StripFM --> LLM["3. call_json(question, context)"]
    LLM --> LLMDetail["POST :11434/api/chat<br/>model: qwen2.5:3b<br/>system: Output ONLY valid JSON<br/>Format: {summary, cards: [{title, summary, details}]}<br/>num_predict: 2000 · temp: 0.1"]

    LLMDetail --> JSONOK{"JSON 解析成功?"}

    JSONOK -->|"否"| KWSearch["回退: keyword_search(question)"]
    KWSearch --> KWSend["send_template_card(第一匹配页面)"]

    JSONOK -->|"是"| ProcessCards["4. 遍历 cards[:8]"]

    ProcessCards --> ForEach["for i, card in enumerate(cards)"]
    ForEach --> Sleep{"i > 0?"}
    Sleep -->|"是"| Wait["time.sleep(0.8)<br/>防止企业微信乱序"]
    Sleep -->|"否"| Normalize["_normalize_details(card.details)<br/>标准化为 [(k,v), ...]"]
    Wait --> Normalize

    Normalize --> ShortSummary{"card.summary 为空?"}
    ShortSummary -->|"是"| CallShort["call_short(title + details)<br/>qwen2.5:3b 一句话摘要"]
    ShortSummary -->|"否"| SendCard["send_template_card(user_id,<br/>title, summary, details,<br/>source_desc='页面名 (i/N)')"]
    CallShort --> SendCard

    SendCard --> MoreCards{"还有卡片?"}
    MoreCards -->|"是"| ForEach
    MoreCards -->|"否"| Overall{"card_data.summary > 100字?"}
    Overall -->|"是"| SendOverall["send_markdown(clean_summary)"]
    Overall -->|"否"| Done["完成"]
    SendOverall --> Done
```

## 七、各阶段对比

| | Phase 0（最初） | Phase 1（基座升级） | Phase 2（当前） |
|---|---|---|---|
| **代码组织** | 3 个独立 repo，2 个单文件脚本 | 统一 `src/knowledge_wiki/` 包 | 不变 |
| **消息分发** | 硬编码 `if/elif/else` | 硬编码 `if/elif/else` | Skill 引擎路由 |
| **技能系统** | 无 | `skill/` 骨架（占位） | 5 技能 + 执行引擎 |
| **LLM 调用** | 散落各处，接口不一 | 统一 `llm/` 包 | 不变 |
| **部署** | systemd 明文密钥 | `.env` + `EnvironmentFile` | 不变 |
| **测试** | 0 | 20 项 | 32 项 |
| **可扩展性** | 加功能 = 加 if/elif 分支 | 加功能 = 加模块 | 加功能 = 加技能目录 |
